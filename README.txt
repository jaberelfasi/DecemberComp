Instructions to run the code:

1- Install Java Development Kit (JDK) (Java SE Development Kit 8u151) if it does not exist.
   http://www.oracle.com/technetwork/pt/java/javase/downloads/jdk8-downloads-2133151.html
2- Make sure the jdk location is inserted in the system environment variable "Path" 
   the java location looks like this: C:\Program Files\Java\jdk1.8.0_152\bin
3- open cmd prompt
4- to run the program navigate to the directory that challenge.java resides in
5- compile challenge.java file as follows: (>javac challenge.java) 
   this will produce the java byte code file (challenge.class) which will need to be run in the next step
6- run the byte code file as follows: (>java challenge)
